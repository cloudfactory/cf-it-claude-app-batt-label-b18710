Battery Labeling

Sign in with your Google account to begin annotating battery chemistry data for Redwood Materials.

Sign in with the @es.cloudfactory.com Google account provided to CloudFactory.

One-time Setup Required

Before the app can work, a Google Sheet needs to be connected as the database. This takes about 5 minutes and only needs to be done once.

1

Create a Google Sheet

Go to sheets.new and create a blank spreadsheet. Name it something like Battery Labeling DB.

2

Enable Google Sheets API

Go to Google Cloud Console → Create a new project → Enable the Google Sheets API and Google Drive API.

3

Create OAuth Credentials

In Cloud Console → APIs & Services → Credentials → Create OAuth 2.0 Client ID → Web Application. Add your app URL as an authorized origin. Copy the Client ID.

4

Enter credentials below

Found in your spreadsheet URL between /d/ and /edit

5

Import your data

Once set up, sign in as admin and use the Import Data button to upload your Excel file. The app will flatten all date sheets into the tasks sheet automatically.
